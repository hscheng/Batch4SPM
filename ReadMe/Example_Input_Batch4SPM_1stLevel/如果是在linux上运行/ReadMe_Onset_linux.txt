第一行是条件的名字，下面的就是Onset，建议先用excel做好数据，再复制到txt中，每个条件下的onset个数可以不一样
示例中有两个run,4个条件，顺序依次如下
run1_cond1,run1_cond2,run1_cond3,run1_cond4,run2_cond2,run2_cond3,run2_cond4

1st row  is the names of conditions across runs,the order follow the sequence:
run1_cond1,run1_cond2,run1_cond3,run1_cond4,run2_cond2,run2_cond3,run2_cond4
